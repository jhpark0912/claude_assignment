# db 구성

## 회원 테이블
- user_id: 유저 id(pk) - string
- username: 유저 이름 - string
- password: 비밀번호 (마스킹) - string
- date_created: 생성일(yyyy-MM-dd)
- date_updated: 수정일(yyyy-MM-dd)

## 회원 포인트 테이블
- id: 포인트 id 값(pk) - int
- user_id: 유저 id (fk) - string
- point: 포인트 (int)
- expired_date: 만료일자(yyyy-MM-dd)
- is_used: 사용여부(boolean)
- date_created: 생성일(yyyy-MM-dd)
- date_updated: 수정일(yyyy-MM-dd)

## 포인트 사용 내역
- id: 사용 내역 id 값(pk) - int
- user_id: 유저 id (fk) - string
- point : 사용한 포인트 (int)
- date_used : 사용된 날자 (yyyy-MM-dd)



# 규칙

## 최대 포인트
- 1일 최대 포인트 10000(1만) 포인트
- 유저 최대 포인트 1000000(100만) 포인트

## 포인트 만료일자
- 포인트 등록일로부터 365일 자정 만료 (expired_date 의 자정 예, 2026-12-31 인 경우 2027-01-01 00:00 만료)

## 포인트 사용규칙
- 포인트는 만료일로부터 가까운 정보 기준으로 사용
- 사용하고자 하는 잔액이 모자라면 사용에 실패한다
- 사용시 여러개의 적립된 포인트를 각 차감한다. 단, 앞서 규칙 대로 만료일 가까운 정보 기준 사용
